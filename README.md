# Экспресс-чекаут Яндекс KIT

Модуль внешнего чекаута Яндекс KIT для Bitrix.

**Версия модуля:** см. `install/index.php` (`MODULE_VERSION` / `MODULE_VERSION_DATE`). На момент последнего обновления README: **0.0.25** (2026-04-07).

## Установка

1. Распакуйте модуль в каталог **`/bitrix/modules/yastore.checkout/`**
2. В админке Bitrix: **Настройки → Настройки продукта → Модули**
3. Найдите модуль **«Экспресс-чекаут Яндекс KIT»** и нажмите **Установить**
4. После установки откройте настройки: **Настройки → Настройки модулей → Экспресс-чекаут Яндекс KIT**

В корень сайта в ходе установки модуля копируется папка `/yastore.checkout/` с точкой входа API (`index.php`). Убедитесь, что папка создана и содержит файлы.

## Настройка

### Токен доступа

В настройках модуля задаётся **токен доступа**. Им подписываются запросы к вашему сайту:

`Authorization: Bearer <TOKEN>`

### Токен API Яндекс KIT

В интерфейсе настроек Яндекс KIT получите **токен API**, вставьте его в настройки модуля и нажмите **Сохранить**.

### Идентификатор товара

В настройках модуля задаются инфоблок каталога и поле идентификатора товара.

Эти настройки нужно менять только если в вашем YML-фиде в поле `<offer id="...">` передаётся не ID элемента инфоблока, а другое значение (символьный код, артикул и т.п.).

### Продажи

Функционал **Продавать все активные товары** используется только в том случае, если на сайте не ведётся количественный учёт. Тогда для покупки доступны все активные товары из фида в пределах значения поля **Количество товара по умолчанию (шт)** на один заказ.

### Статусы заказов

На данной вкладке необходимо убедиться, что корректно заполнены статусы заказов в соответствии со входящим статусом заказа.

### Автоматизация

Функционал **автоматической отмены** заказа отслеживает изменение статуса заказа и при наступлении выбранного статуса отправляет сигнал отмены в Яндекс KIT.

**Автоматическое завершение доставки** используется в случае самовывоза или самостоятельной доставки заказа силами продавца.

### Кнопка «Купить в 1 клик»

Функционал автоматически отображает кнопку «Купить в 1 клик» для перехода на оформление заказа в Яндекс KIT со страницы корзины.

Необходимо указать адрес страницы корзины, а также элемент вёрстки, после которого будет установлена кнопка. С помощью CSS и селектора можно стилизовать кнопку под сайт.

### Торговые предложения

Если товары имеют характеристики (цвет, размер и т.д.) и связаны между собой торговыми предложениями, необходимо выбрать инфоблок и свойства, отличающие варианты друг от друга. Параметр «Свойство цвет» используется для графического отображения цветов товаров на странице оформления заказа.
